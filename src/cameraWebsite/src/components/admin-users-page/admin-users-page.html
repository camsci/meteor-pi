Manage users

Key

Login
Check this box to allow the user to log in. In effect this can be used to disable a user temporarily without actually deleting the user concerned.
Admin
Admin users can access this page, the camera status update and the import controls. Only check this box for users you trust, as any users with this role can remove your admin permissions.
Import
This option allows users to connect from another node in the camera network and send data to this node. Typically these users won't be people, they'll be used for software on different machines when connecting to this one. Without this permission the user can't push data to here, so it acts as a way to (temporarily) disable information import from this source.
